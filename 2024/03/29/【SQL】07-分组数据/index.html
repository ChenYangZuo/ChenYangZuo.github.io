【SQL】07-分组数据

创建分组

GROUP BY子句必须出现在WHERE子句之后，ORDER BY子句之前

过滤分组

WHERE过滤指定的是行而不是分组；HAVING可以过滤分组，所有类型的WHERE子句都可以用HAVING来替代

WHERE在数据分组前进行过滤，HAVING在数据分组后进行过滤，WHERE排除的行不在分组内，可能改变计算值

SELECT子句顺序

子句 说明 必须使用
SELECT 返回数据 是
FROM 从中检索数据的表 仅在从表选择数据时使用
WHERE 行级过滤 否
GROUP BY 分组说明 仅在按组计算聚集时使用
HAVING 组级过滤 否
ORDER BY 输出排序顺序 否
